## 🔧 CUSTOMIZATION GUIDE

### Change Colors
Edit `/styles.css` lines 9-16:
```css
:root {
  --primary-color: #6B8E7A;   /* Change this */
  --sage-green: #6B8E7A;      /* And this */
  --bg-light: #F5F5F0;        /* Background */
  --text-dark: #1f2937;       /* Text color */
  --text-gray: #6b7280;       /* Secondary text */
}
```

### Change Content
1. Open any `.html` file in text editor
2. Find the text you want to change
3. Edit and save
4. Refresh browser to see changes

### Change Images
1. Find image URL in HTML (look for `img src="..."`)
2. Replace with your own image URL
3. Or use different Unsplash image
4. Save and refresh

### Add New Page
1. Duplicate existing HTML file
2. Rename it (e.g., `blog.html`)
3. Update content
4. Add link in navigation (all pages)
5. Upload new file
