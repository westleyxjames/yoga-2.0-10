## 🎨 Customization

### Change Colors:
Edit `styles.css` line 9-16 (CSS variables):
```css
:root {
  --primary-color: #6B8E7A;  /* Change this */
  --sage-green: #6B8E7A;     /* And this */
  --bg-light: #F5F5F0;       /* Background color */
}
```

### Update Content:
- Open any `.html` file in text editor
- Edit text directly
- Save and refresh browser

### Add/Change Images:
- Replace Unsplash URLs with your own
- Or use Unsplash API for different images
